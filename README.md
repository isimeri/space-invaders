## Space Invaders -- [[live]](https://isimeri.github.io/space-invaders/)

The classic game of Space Invaders.  
In order to succeed in defending Earth and be victorious, you must defeat all the alien invaders before they reach the Earth's surface.  
Use the **left** and **right arrow** keys on your keyboard to move the hero. Press **Z** to shoot your laser at the enemy.  

![Image Chest](https://cdn.imgchest.com/files/6e868a8ce06b.png)

There are a few interactive elements on the screen as well:
- The **board width slider** lets you adjust the board width between **15** and **20**. This in turn affects how many rows of enemies there will be and the maximum number of enemies per row. The slider basically acts like a difficulty setting.  
The slider can't be changed while the game is running. So, either finish your current game, or press the **Reset** button to end it.
- The **Start** button will start the game with the adjusted settings (see slider above)
- The **Restart** button will start the game with the same settings from the previous game.
- The **Reset** button will end the current game and reset the board. Basically acts like a page refresh.

## How to run locally
1. Clone the repo
2. Run these commands, in this order, inside the root folder:  
  2a. `npm install`  
  2b. `npm run build`  
  2c. `npm run dev`
3. Now the app should be accessible at *`localhost:8080`*
